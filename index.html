📊 Notions économiques clés Fondamentaux
PIB = valeur totale des biens/services produits dans un pays. Croissance = hausse du PIB. Récession = 2 trimestres consécutifs de baisse. Croissance zone euro ≈ 0,8% en 2024.
Inflation = hausse générale des prix. Mesurée par l'IPC (Indice des Prix à la Consommation). Objectif BCE = 2%. Déflation = baisse des prix (dangereux car reporte achats).
La BCE (Banque Centrale Européenne) gère les taux d'intérêt en zone euro. Taux élevé = freine inflation mais freine aussi la croissance. FED = banque centrale américaine.
OMC = Organisation Mondiale du Commerce. Balance commerciale = exports - imports. France : déficit commercial persistant. Protectionnisme en hausse (Trump, taxes douanières USA-Chine).
Taux de chômage France ≈ 7,3% fin 2024. France Travail (ex-Pôle Emploi) depuis jan. 2024. Plein emploi ≈ objectif de 5%. Chômage structurel vs conjoncturel.
Dette France ≈ 110% du PIB. Critères de Maastricht : dette max 60% PIB, déficit max 3%. France en procédure de déficit excessif depuis 2024.

⚠️ Piège classique : 35 questions d'expression (orthographe, grammaire, conjugaison, vocab) + 15 questions de compréhension de texte. Évite les fautes bêtes d'accord !
✅ Accords essentiels — Les règles à ne pas rater
S'accorde avec le COD si le COD est placé AVANT le verbe.
✅ "Les lettres qu'il a écrites" (COD "lettres" avant = accord)
✅ "Il a écrit des lettres" (COD après = pas d'accord)
Astuce : demande "quoi ?" ou "qui ?" après le verbe → si la réponse est avant → accorde.
S'accorde toujours avec le sujet.
✅ "Elles sont parties" / "Ils sont arrivés"
Verbes avec être : naître, mourir, aller, venir, partir, arriver, entrer, sortir, tomber, rester, monter, descendre... et tous les verbes pronominaux.
🔸 Ni...ni → pluriel : "Ni Paul ni Pierre ne sont venus."
🔸 La plupart de → pluriel : "La plupart des étudiants réussissent."
🔸 Chacun / chacune → singulier : "Chacun fait de son mieux."
🔸 Collectif + complément : "Une équipe de joueurs est/sont prêts" → accord avec le sens.
🔸 C'est / ce sont : "C'est eux" ou "Ce sont eux" (les deux admis).
S'accorde avec le nom auquel il se rapporte (genre + nombre).
🔸 Adjectif de couleur composé = invariable : "des robes bleu clair"
🔸 Couleur nom commun = invariable : "des yeux marron", "des tapis orange"
🔸 Demi avant nom = invariable : "demi-heure" ; après = accord : "deux heures et demie"
🔤 Pronoms — Tableau de bord
COD (répond à "qui ?" / "quoi ?") : me, te, le, la, nous, vous, les
✅ "Je le vois" (le = Paul)

COI (répond à "à qui ?" / "à quoi ?") : me, te, lui, nous, vous, leur
✅ "Je lui parle" (lui = à Paul)

🔸 Y = remplace lieu ou complément introduit par "à" (chose)
🔸 En = remplace complément introduit par "de"
Qui = sujet : "L'homme qui parle est grand."
Que = COD : "Le livre que tu lis est intéressant."
Dont = complément introduit par "de" : "L'ami dont je te parle."
Où = lieu ou temps : "La ville où il vit / Le jour où il est arrivé."
Lequel/laquelle/lesquels = après préposition : "L'outil avec lequel il travaille."
Démonstratifs : celui, celle, ceux, celles (+ ci/là)
Possessifs : le mien, le tien, le sien, le nôtre, le vôtre, le leur
⚠️ "Notre" (adj.) vs "le nôtre" (pronom) — accent circonflexe sur le pronom !
⏰ Conjugaison — Temps à maîtriser absolument
Utilisé après : "bien que", "pour que", "quoique", "bien que", "à condition que", "vouloir que"...

Être : que je sois, que tu sois, qu'il soit, que nous soyons
Avoir : que j'aie, que tu aies, qu'il ait, que nous ayons
Aller : que j'aille, que nous allions
Faire : que je fasse, que nous fassions
Pouvoir : que je puisse
Savoir : que je sache
Formation : radical du futur + terminaisons imparfait (-ais, -ais, -ait, -ions, -iez, -aient)
✅ "Il viendrait" = viendr- + -ait

Usages : hypothèse, politesse, information non confirmée
"Le président aurait signé l'accord." (information non vérifiée = conditionnel journalistique)
🔸 a / à : "il a" (avoir) vs "à" (préposition) → remplacer par "avait"
🔸 ou / où : "ou" = liaison ; "où" = lieu/temps
🔸 et / est : "et" = liaison ; "est" = être → remplacer par "était"
🔸 ce / se : "ce livre" (démonstratif) / "il se lève" (pronom réfléchi)
🔸 ses / ces / c'est / s'est : bien distinguer selon le sens
🔸 leur / leurs : "leur" adj. = invariable si modifie un singulier ; "leurs" si pluriel

📐 Grammaire anglaise — Points clés
a/an = première mention, chose non spécifique. "an" devant voyelle sonore.
✅ "an hour" (h muet), "a university" (se dit "you-")
the = chose spécifique, déjà mentionnée, unique
ø = pas d'article devant : noms propres, matières, repas, pays (sauf the UK, the USA, the Netherlands)
Present Simple = habitudes, vérités générales
Present Continuous = action en cours (be + -ing)
Present Perfect = action passée avec résultat présent (have + PP)
"I have just finished" / "She has worked here for 3 years."
Past Simple = action terminée à un moment précis
⚠️ Pas de "since/for" avec le past simple en anglais britannique
1 syllabe : big → bigger → the biggest
2 syllabes en -y : happy → happier → the happiest
3+ syllabes : important → more important → the most important
Irréguliers : good/better/the best — bad/worse/the worst — far/farther/the farthest
can/could = capacité, possibilité
may/might = permission, éventualité (might = moins certain)
must = obligation (interne) / "must not" = interdiction
have to / need to = obligation (externe)
should/ought to = conseil, recommandation
would = conditionnel, politesse ("Would you like...?")
🔸 much/many : much = indénombrable (much money) ; many = dénombrable (many people)
🔸 few/a few : few = presque aucun (négatif) ; a few = quelques-uns (positif)
🔸 little/a little : idem avec indénombrables
🔸 make vs do : make = créer (make a decision) ; do = effectuer (do homework)
🔸 say vs tell : say + parole ; tell + personne ("tell me", "say something")

➕ Calcul algébrique — Rappels express Maths
(a + b)² = a² + 2ab + b²
(a - b)² = a² - 2ab + b²
(a + b)(a - b) = a² - b²
Astuce : mémorise-les, elles servent souvent à factoriser ou développer vite.
1er degré : ax + b = 0 → x = -b/a

2ème degré : ax² + bx + c = 0
Δ = b² - 4ac
  • Δ > 0 → 2 solutions : x = (-b ± √Δ) / 2a
  • Δ = 0 → 1 solution : x = -b / 2a
  • Δ < 0 → pas de solution réelle
Taux de variation = (V_final - V_initial) / V_initial × 100
🔸 Augmentation de 20% puis baisse de 20% ≠ retour à la valeur initiale !
(×1,2) puis (×0,8) = ×0,96 → perte de 4%

🔸 Coefficient multiplicateur : +30% → ×1,3 ; -15% → ×0,85
🔸 Pour trouver la valeur initiale : V_initial = V_final / coeff multiplicateur

🔢 Suites & Statistiques
Chaque terme = terme précédent + raison (r)
uₙ = u₀ + n × r
Somme des n+1 premiers termes : S = (n+1) × (u₀ + uₙ) / 2
Exemple : 2, 5, 8, 11... → r = 3, u₀ = 2, u₁₀ = 2 + 10×3 = 32
Chaque terme = terme précédent × raison (q)
uₙ = u₀ × qⁿ
Exemple : 3, 6, 12, 24... → q = 2, u₀ = 3, u₅ = 3 × 2⁵ = 96
Moyenne = somme des valeurs / nombre de valeurs
Médiane = valeur centrale (série classée par ordre) — si n pair : moyenne des 2 valeurs centrales
Mode = valeur la plus fréquente
Étendue = max - min
Variance = moyenne des carrés des écarts à la moyenne
P(A) = nombre de cas favorables / nombre de cas possibles
🔸 P(A) toujours entre 0 et 1
🔸 P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
🔸 Si A et B indépendants : P(A ∩ B) = P(A) × P(B)
🔸 Complémentaire : P(Ā) = 1 - P(A)
🧩 Raisonnement logique — Types de questions
Cherche : +n, ×n, alternance, carrés, Fibonacci...
Exemples :
🔸 1, 4, 9, 16, 25... → carrés parfaits (n²)
🔸 1, 1, 2, 3, 5, 8, 13... → Fibonacci (somme des 2 précédents)
🔸 2, 6, 18, 54... → ×3 (géométrique de raison 3)
🔸 1, 2, 4, 7, 11, 16... → différences : +1, +2, +3, +4, +5
Méthode : 1) Lis les deux prémisses. 2) Déduis uniquement ce qui est logiquement nécessaire. 3) Ne rajoute pas de connaissance externe.

Exemple :
"Tous les A sont B. X est A. Donc X est B." ✅
"Certains A sont B. X est A. Donc X est B." ❌ (pas forcément !)
